# obsi-math

Mathematical plugin for Obsidian built with TypeScript.

## Usage

### Function analysis

Write a function inside an `obs-math` code block:

````
```obs-math
f(x)=x^2
````

```

Obsi Math analyzes the function and displays mathematical information such as roots, critical points, and other relevant results.

### Systems of equations

Write a system inside an `obs-sistema` code block:

```

```obs-sistema
x+y=5
2x-y=1
```

```

Obsi Math solves the system and displays the corresponding mathematical information.

## Features

- Mathematical expression analysis
- Root detection
- Critical point analysis
- Linear system solving
- KaTeX rendering
- Obsidian integration
```
